## INVENTORY PROGRAM
This is a simple **inventory system** that allows you to add, read, update and delete products. 
It also allows you to save the inventory in a csv file and read it later.
1. First you have to had **python last version** in your computer, then you have to clone the repository and run the menu.py file, or download the file and open it in visual studio code, for better.
2. The inventory system is implemented in Python and it is a simple command line application. The inventory is stored in a list of dictionaries, where each dictionary represents a product with its name, price, amount and total price.
3. The menu is implemented in the menu.py file, where the user can select the option to add, read, update, delete, show statistics, save and load the inventory.
4. The validations are implemented in the validations.py file, where the user is asked to input the product name, price and amount, and the data is validated to avoid errors.
5. The services are implemented in the services.py file, where the functions to add, read, update  and delete products are defined, as well as the functions to save and read the inventory in a csv file.
